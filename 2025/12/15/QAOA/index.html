Quantum Approximate Optimization Algorithm (QAOA)

Problem Setting

Let z=z1z2…znz=z_1 z_2 \ldots z_nz=z1​z2​…zn​ be a string of nnn bits, where each bit zi∈{0,1}z_i \in \{0, 1\}zi​∈{0,1}. We define a cost function C(z)C(z)C(z) that assigns a real value to each bit string. The goal is to find the bit string z∗z^*z∗ that maximizes the cost function:

z∗=arg⁡max⁡z∈{0,1}nC(z)=arg⁡max⁡z∈{0,1}n∑αCα(z)z^* = \arg\max_{z \in \{0,1\}^n} C(z)=\arg\max_{z \in \{0,1\}^n} \sum_{\alpha} C_{\alpha}(z) z∗=argz∈{0,1}nmax​C(z)=argz∈{0,1}nmax​α∑​Cα​(z)

where Cα(z)C_{\alpha}(z)Cα​(z) are indicator functions 1clause α is satisfied by z\mathbf{1}_{\text{clause } \alpha \text{ is satisfied by } z}1clause α is satisfied by z​.

Hamiltonian Representation

We can represent the cost function as a Hamiltonian operator acting on a quantum state. The Hamiltonian HCH_CHC​ corresponding to the cost function C(z)C(z)C(z) is given by:

HC=∑αHCαH_C = \sum_{\alpha} H_{C_{\alpha}} HC​=α∑​HCα​​

where each HCαH_{C_{\alpha}}HCα​​ is a diagonal operator in the computational basis defined as:

HCα∣z⟩=Cα(z)∣z⟩H_{C_{\alpha}} \ket{z}= C_{\alpha}(z) \ket{z} HCα​​∣z⟩=Cα​(z)∣z⟩

Each HCαH_{C_{\alpha}}HCα​​ is Hermitian, so is HCH_CHC​ and HC∣z⟩=C(z)∣z⟩H_C \ket{z} = C(z) \ket{z}HC​∣z⟩=C(z)∣z⟩.

We now seek to find the maximum eigenvalue λ∗\lambda^*λ∗ of HCH_CHC​ and the corresponding eigenstate ∣z∗⟩\ket{z^*}∣z∗⟩.

What we can learn from Score Based Model (SBM)

In EBM, we define a probability distribution via the energy function (Hamiltonian):

p(z)=exp⁡(−E(z))Z,Z=∫exp⁡(−E(z))dzp(z) = \frac{\exp(-E(z))}{Z}, Z = \int \exp(-E(z)) dz p(z)=Zexp(−E(z))​,Z=∫exp(−E(z))dz

consider a parameterized model p(z;θ)p(z; \theta)p(z;θ), we can learn the parameters θ\thetaθ by learning the score function ∇zlog⁡p(z;θ)\nabla_z \log p(z; \theta)∇z​logp(z;θ).

LSM(θ)=Epdata(x)[Tr(∇xsθ(x))+12∥sθ(x)∥22]\boxed{ \mathcal{L}_{\mathrm{SM}}(\theta) = \mathbb{E}_{p_{data}(x)} \left[\mathrm{Tr}(\nabla_x s_{\theta}(x)) + \frac{1}{2} \| s_{\theta}(x) \|_2^2\right] } LSM​(θ)=Epdata​(x)​[Tr(∇x​sθ​(x))+21​∥sθ​(x)∥22​]​

where sθ(x)=∇xlog⁡p(x;θ)s_{\theta}(x)= \nabla_x \log p(x; \theta)sθ​(x)=∇x​logp(x;θ) is the score function parameterized by θ\thetaθ.

In QAOA’s scenario, the energy function is fixed and we want to find the state ∣z∗⟩\ket{z^*}∣z∗⟩ that maximizes the energy.

Let’s define a quantum state ∣ψ(θ)⟩=U(θ)∣0⟩\ket{\psi(\theta)} = U(\theta) \ket{0}∣ψ(θ)⟩=U(θ)∣0⟩ parameterized by θ\thetaθ.

We can measure:

⟨HC⟩θ=⟨ψ(θ)∣HC∣ψ(θ)⟩\langle H_C \rangle_{\theta} = \bra{\psi(\theta)} H_C \ket{\psi(\theta)} ⟨HC​⟩θ​=⟨ψ(θ)∣HC​∣ψ(θ)⟩

We want to maximize:

J(θ)=⟨ψ(θ)∣HC∣ψ(θ)⟩=∑z∈{0,1}nP(z∣θ)C(z)J(\theta) = \langle \psi(\theta) | H_C | \psi(\theta) \rangle = \sum_{z \in \{0,1\}^n} P(z|\theta) C(z) J(θ)=⟨ψ(θ)∣HC​∣ψ(θ)⟩=z∈{0,1}n∑​P(z∣θ)C(z)

where P(z∣θ)=∣⟨z∣ψ(θ)⟩∣2P(z|\theta) = |\langle z | \psi(\theta) \rangle|^2P(z∣θ)=∣⟨z∣ψ(θ)⟩∣2 is the probability of measuring the state ∣z⟩\ket{z}∣z⟩ given the parameter θ\thetaθ.

In the first glance, we try to estimiate the log gradient:

∇θJ(θ)=∇θ∑zP(z∣θ)C(z)=∑zC(z)∇θP(z∣θ)=∑zC(z)P(z∣θ)∇θlog⁡P(z∣θ)=Ez∼P(z∣θ)[C(z)∇θlog⁡P(z∣θ)]\begin{aligned} \nabla_\theta J(\theta) &= \nabla_\theta \sum_{z} P(z|\theta) C(z) \\ &= \sum_{z} C(z) \nabla_\theta P(z|\theta) \\ &= \sum_{z} C(z) P(z|\theta) \nabla_\theta \log P(z|\theta) \\ &= \mathbb{E}_{z \sim P(z|\theta)} \left[ C(z) \nabla_\theta \log P(z|\theta) \right] \end{aligned} ∇θ​J(θ)​=∇θ​z∑​P(z∣θ)C(z)=z∑​C(z)∇θ​P(z∣θ)=z∑​C(z)P(z∣θ)∇θ​logP(z∣θ)=Ez∼P(z∣θ)​[C(z)∇θ​logP(z∣θ)]​

This is similar to the REINFORCE learning gradient method with reward C(z)C(z)C(z).
However, this gradient is not calculable directly on quantum computer since log⁡P(z∣θ)\log P(z|\theta)logP(z∣θ) cannot be computed efficiently. (2n2^n2n terms). And it does not use the result of measure J(θ)J(\theta)J(θ) directly.

On the other hand, if we sampling through

lim⁡ϵ→0J(θ+ϵ)−J(θ)ϵ=∇θJ(θ)\lim_{\epsilon \to 0} \frac{J(\theta + \epsilon) - J(\theta)}{\epsilon} = \nabla_\theta J(\theta) ϵ→0lim​ϵJ(θ+ϵ)−J(θ)​=∇θ​J(θ)

This suffers from high variance issue. The high variance is due to the high dimensionality of the output space.

Parameter Shift Rule

Remember that in Denoising Score Matching, we can avoid calculating the score function directly by introducing noise and learning to denoise, then the score function can be computed directly. And under Gaussian noise, the score function has a closed form.

This inspires us to introduce a particular parameterization of U(θ)U(\theta)U(θ) in QAOA such that we can compute the gradient of J(θ)J(\theta)J(θ) directly.

For U(θ)=e−iθ2GU(\theta) = e^{-i \frac{\theta}{2} G}U(θ)=e−i2θ​G, where GGG is a Hermitian operator (generator), we have:

ddθU(θ)=−i2GU(θ)=−i2U(θ)G\frac{d}{d\theta} U(\theta) = -\frac{i}{2} G U(\theta) = - \frac{i}{2} U(\theta) G dθd​U(θ)=−2i​GU(θ)=−2i​U(θ)G

this uses the property that GGG commutes with U(θ)U(\theta)U(θ).

Thus,

∇θJ(θ)=∇θ⟨ψ0∣U†(θ)HCU(θ)∣ψ0⟩=⟨ψ0∣(ddθU†(θ))HCU(θ)∣ψ0⟩+⟨ψ0∣U†(θ)HC(ddθU(θ))∣ψ0⟩=⟨ψ0∣(i2U†(θ)G)HCU(θ)∣ψ0⟩+⟨ψ0∣U†(θ)HC(−i2GU(θ))∣ψ0⟩=i2⟨ψ(θ)∣[G,HC]∣ψ(θ)⟩\begin{aligned} \nabla_\theta J(\theta) &= \nabla_\theta \langle \psi_0 | U^\dagger(\theta) H_C U(\theta) | \psi_0 \rangle \\ = & \langle \psi_0 | \left( \frac{d}{d\theta} U^\dagger(\theta) \right) H_C U(\theta) | \psi_0 \rangle + \langle \psi_0 | U^\dagger(\theta) H_C \left( \frac{d}{d\theta} U(\theta) \right) | \psi_0 \rangle \\ = & \langle \psi_0 | \left( \frac{i}{2} U^\dagger(\theta) G \right) H_C U(\theta) | \psi_0 \rangle + \langle \psi_0 | U^\dagger(\theta) H_C \left( -\frac{i}{2} G U(\theta) \right) | \psi_0 \rangle \\ = & \frac{i}{2} \langle \psi(\theta) | [G, H_C] | \psi(\theta) \rangle \end{aligned} ∇θ​J(θ)===​=∇θ​⟨ψ0​∣U†(θ)HC​U(θ)∣ψ0​⟩⟨ψ0​∣(dθd​U†(θ))HC​U(θ)∣ψ0​⟩+⟨ψ0​∣U†(θ)HC​(dθd​U(θ))∣ψ0​⟩⟨ψ0​∣(2i​U†(θ)G)HC​U(θ)∣ψ0​⟩+⟨ψ0​∣U†(θ)HC​(−2i​GU(θ))∣ψ0​⟩2i​⟨ψ(θ)∣[G,HC​]∣ψ(θ)⟩​

where [G,HC]=GHC−HCG[G, H_C] = G H_C - H_C G[G,HC​]=GHC​−HC​G is the commutator of GGG and HCH_CHC​, ψ(θ)=U(θ)∣ψ0⟩\psi(\theta) = U(\theta) \ket{\psi_0}ψ(θ)=U(θ)∣ψ0​⟩.

Now, note that

H+=(e−iπ4G)†HC(e−iπ4G)=eiπ4GHCe−iπ4G=12(I+iG)HC(I−iG)=12(HC−iHCG+iGHC+GHCG)H−=(eiπ4G)†HC(eiπ4G)=e−iπ4GHCeiπ4G=12(HC+iHCG−iGHC+GHCG)\begin{aligned} H_+ &= (e^{-i \frac{\pi}{4} G})^\dagger H_C (e^{-i \frac{\pi}{4} G}) = e^{i \frac{\pi}{4} G} H_C e^{-i \frac{\pi}{4} G}\\ &= \frac{1}{2} (I + iG) H_C (I - iG) \\ &= \frac{1}{2} (H_C - i H_C G + i G H_C + G H_C G)\\ H_- &= (e^{i \frac{\pi}{4} G})^\dagger H_C (e^{i \frac{\pi}{4} G}) = e^{-i \frac{\pi}{4} G} H_C e^{i \frac{\pi}{4} G}\\ &= \frac{1}{2} (H_C + i H_C G - i G H_C + G H_C G) \end{aligned} H+​H−​​=(e−i4π​G)†HC​(e−i4π​G)=ei4π​GHC​e−i4π​G=21​(I+iG)HC​(I−iG)=21​(HC​−iHC​G+iGHC​+GHC​G)=(ei4π​G)†HC​(ei4π​G)=e−i4π​GHC​ei4π​G=21​(HC​+iHC​G−iGHC​+GHC​G)​

Thus,

H+−H−=i(GHC−HCG)=i[G,HC]H_+ - H_- = i (G H_C - H_C G) = i [G, H_C] H+​−H−​=i(GHC​−HC​G)=i[G,HC​]

Therefore, we have:

∇θJ(θ)=12⟨ψ(θ)∣(H+−H−)∣ψ(θ)⟩=12(J(θ+π2)−J(θ−π2))\nabla_\theta J(\theta) = \frac{1}{2} \langle \psi(\theta) | (H_+ - H_-) | \psi(\theta) \rangle = \frac{1}{2} \left( J(\theta + \frac{\pi}{2}) - J(\theta - \frac{\pi}{2}) \right) ∇θ​J(θ)=21​⟨ψ(θ)∣(H+​−H−​)∣ψ(θ)⟩=21​(J(θ+2π​)−J(θ−2π​))

This means we can compute the gradient of J(θ)J(\theta)J(θ) by evaluating JJJ at two shifted parameter values, avoiding the need to compute the score function directly.

Parameterized Quantum Circuit in QAOA

In QAOA, the parameterized quantum circuit is constructed using two types of unitary operators: the cost unitary UC(γ)U_C(\gamma)UC​(γ) and the mixing unitary UB(β)U_B(\beta)UB​(β). The overall circuit for ppp layers is given by:

U(γ,β)=UB(βp)UC(γp)…UB(β1)UC(γ1)U(\boldsymbol{\gamma}, \boldsymbol{\beta}) = U_B(\beta_p) U_C(\gamma_p) \ldots U_B(\beta_1) U_C(\gamma_1) U(γ,β)=UB​(βp​)UC​(γp​)…UB​(β1​)UC​(γ1​)

The cost unitary is defined as:

UC(γ)=e−iγHCU_C(\gamma) = e^{-i \gamma H_C} UC​(γ)=e−iγHC​

and the mixing unitary is defined as:

UB(β)=e−iβHBU_B(\beta) = e^{-i \beta H_B} UB​(β)=e−iβHB​

where HB=∑i=1nXiH_B = \sum_{i=1}^n X_iHB​=∑i=1n​Xi​ with Xi=I⊗(i−1)⊗X⊗I⊗(n−i)X_i = I^{\otimes (i-1)} \otimes X \otimes I^{\otimes (n-i)}Xi​=I⊗(i−1)⊗X⊗I⊗(n−i) being the Pauli-X operator acting on qubit iii.

The initial state is typically chosen as the uniform superposition state:

∣ψ0⟩=12n∑z∈{0,1}n∣z⟩\ket{\psi_0} = \frac{1}{\sqrt{2^n}} \sum_{z \in \{0,1\}^n} \ket{z} ∣ψ0​⟩=2n​1​z∈{0,1}n∑​∣z⟩

which is the ground state of the mixing Hamiltonian HBH_BHB​.

The Adiabatic Theorem suggests that if we vary the parameters γ\boldsymbol{\gamma}γ and β\boldsymbol{\beta}β slowly enough, the system will remain in its ground state, allowing us to approximate the optimal solution to the original combinatorial optimization problem.

H(t)=(1−tT)HB+tTHCH(t) = (1 - \frac{t}{T}) H_B + \frac{t}{T} H_C H(t)=(1−Tt​)HB​+Tt​HC​

The time evolution operator can be approximated using the Trotter-Suzuki decomposition:

e−i(A+B)t≈(e−iAtne−iBtn)ne^{-i (A+B) t} \approx \left( e^{-i A \frac{t}{n}} e^{-i B \frac{t}{n}} \right)^n e−i(A+B)t≈(e−iAnt​e−iBnt​)n

Algorithm Summary

  1. Initialize the quantum state ∣ψ0⟩\ket{\psi_0}∣ψ0​⟩ as the uniform superposition state.
  2. Choose the number of layers ppp and initialize the parameters γ,β\boldsymbol{\gamma}, \boldsymbol{\beta}γ,β.
  3. Construct the parameterized quantum circuit U(γ,β)U(\boldsymbol{\gamma}, \boldsymbol{\beta})U(γ,β).
  4. Measure the expectation value J(γ,β)=⟨ψ(γ,β)∣HC∣ψ(γ,β)⟩J(\boldsymbol{\gamma}, \boldsymbol{\beta}) = \langle \psi(\boldsymbol{\gamma}, \boldsymbol{\beta}) | H_C | \psi(\boldsymbol{\gamma}, \boldsymbol{\beta}) \rangleJ(γ,β)=⟨ψ(γ,β)∣HC​∣ψ(γ,β)⟩.
  5. Use the parameter shift rule to compute the gradients ∇γJ\nabla_{\boldsymbol{\gamma}} J∇γ​J and ∇βJ\nabla_{\boldsymbol{\beta}} J∇β​J.
  6. Update the parameters γ,β\boldsymbol{\gamma}, \boldsymbol{\beta}γ,β using a classical optimization algorithm (e.g., gradient descent).
  7. Repeat steps 3-6 until convergence or a stopping criterion is met.
  8. Measure the final state to obtain a bit string zzz that approximates the optimal solution to the original problem.
