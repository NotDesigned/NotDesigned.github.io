子模函数 (Submodular Function)

一个函数 f:2N→Rf: 2^N \to \mathbb{R}f:2N→R 被称为子模函数，如果对于任意的 A⊆B⊆NA \subseteq B \subseteq NA⊆B⊆N 和 x∈N∖Bx \in N \setminus Bx∈N∖B，都有以下不等式成立：

f(A∪{x})−f(A)≥f(B∪{x})−f(B)f(A \cup \{x\}) - f(A) \geq f(B \cup \{x\}) - f(B) f(A∪{x})−f(A)≥f(B∪{x})−f(B)

一个等价的定义是，如果对于任意的 A⊆BA \subseteq BA⊆B 和 x∈N∖Bx \in N \setminus Bx∈N∖B，都有：

f(A∪{x})−f(A)≥f(B∪{x})−f(B)f(A \cup \{x\}) - f(A) \geq f(B \cup \{x\}) - f(B) f(A∪{x})−f(A)≥f(B∪{x})−f(B)

第一个不等式可以被解释为“边际收益递减”，即添加一个元素到集合 AAA 的收益大于或等于添加同样的元素到更大的集合 BBB 的收益。

几个常见的子模函数示例包括：

  • 令图 G=(V,E)G = (V, E)G=(V,E)，定义函数 f(S)=∣E(S,Sˉ)∣f(S) = |E(S, \bar S)|f(S)=∣E(S,Sˉ)∣，则fff 是子模的，其中 E(S,Sˉ)E(S, \bar S)E(S,Sˉ) 表示集合 SSS 和其补集 Sˉ\bar SSˉ 之间的边集。
  • 令图 G=(U,V,E)G = (U, V, E)G=(U,V,E) 为一个二分图，∣U∣=n|U| = n∣U∣=n。 对于所有 S⊆US \subseteq US⊆U，定义 f(S)=∣N(S)∣f(S) = |N(S)|f(S)=∣N(S)∣，其中 N(S)N(S)N(S) 是 SSS 的邻居节点集合，则 fff 是子模的。fff也是单调的。
  • 令 x1,x2,⋯ ,xnx_1,x_2,\cdots, x_nx1​,x2​,⋯,xn​ 是 nnn 个离散随机变量。对任意 A⊆[n]A \subseteq [n]A⊆[n]，定义 f(A)=H(XA)f(A) = H(X_A)f(A)=H(XA​)，其中 H(XA)H(X_A)H(XA​) 是 XAX_AXA​ 的熵，则 fff 是子模的。

Lovász拓展 (Lovász Extension)

对于函数 f:2N→Rf: 2^N \to \mathbb{R}f:2N→R，其Lovász拓展定义为：

f^(x)=Eλ∼Uniform(0,1)[f({i∈N:xi≥λ})]\hat{f}(x) = \mathbb{E}_{\lambda \sim \text{Uniform}(0,1)}[f(\{i \in N : x_i \geq \lambda\})] f^​(x)=Eλ∼Uniform(0,1)​[f({i∈N:xi​≥λ})]

其中 x=(x1,x2,…,xn)∈Rnx = (x_1, x_2, \ldots, x_n)\in \mathbb{R}^nx=(x1​,x2​,…,xn​)∈Rn。

Lovász拓展被称为拓展，因为它保留了离散点集上的函数值。注意到，对于离散点z∈{0,1}n,λ∈[0,1]z\in \{0,1\}^n, \lambda \in [0, 1]z∈{0,1}n,λ∈[0,1]，都有：{i∣zi≥λ}={i∣zi=1}=S\{ i | z_i\geq \lambda\} = \{ i | z_i = 1 \} = S{i∣zi​≥λ}={i∣zi​=1}=S。所以 f^\hat ff^​与 fff 在离散点集上是相同的。

另一个等价的构造定义

将向量 xxx 的分量按降序重新排列为 xσ(1)≥xσ(2)≥…≥xσ(n)x_{\sigma(1)} \geq x_{\sigma(2)} \geq \ldots \geq x_{\sigma(n)}xσ(1)​≥xσ(2)​≥…≥xσ(n)​，然后定义集合序列 Si={σ(1),σ(2),…,σ(i)}={i∈N:xi≥xσ(i)}S_i = \{ \sigma(1), \sigma(2), \ldots, \sigma(i) \} = \{i \in N : x_i \geq x_{\sigma(i)}\}Si​={σ(1),σ(2),…,σ(i)}={i∈N:xi​≥xσ(i)​}。

则：

f^(x)=∑i=1nf(Si)⋅(xσ(i)−xσ(i+1))\hat{f}(x) = \sum_{i=1}^n f(S_{i}) \cdot (x_{\sigma(i)} - x_{\sigma(i+1)}) f^​(x)=i=1∑n​f(Si​)⋅(xσ(i)​−xσ(i+1)​)

这个定义可以看作是对函数 fff 在每个集合 SiS_iSi​ 上的值进行加权平均，其中权重是相邻元素之间的差值。
