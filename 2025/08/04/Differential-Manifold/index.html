Partition of Unity

Partitions of unity are one of the most powerful tools in the theory of smooth manifolds. They provide a way to smoothly “glue together” local constructions into global ones. For example, if we can define an object (like a function, a metric, or a differential form) on each chart of an atlas, a partition of unity allows us to combine these local objects into a single, globally defined smooth object on the entire manifold. The fundamental building blocks for partitions of unity are smooth bump functions.

Bump Functions

A bump function on a manifold MMM is a smooth function λ:M→R\lambda: M \to \mathbb{R}λ:M→R that is equal to 1 on some specified compact set and is zero outside of a slightly larger open set containing it. The existence of such functions is a cornerstone of analysis on manifolds, distinguishing smooth manifolds from, for example, analytic manifolds where such functions do not exist.

First, let’s establish their existence on Rn\mathbb{R}^nRn. Consider the function ψ:R→R\psi: \mathbb{R} \to \mathbb{R}ψ:R→R defined by:

ψ(t)={e−1/tif t>00if t≤0\psi(t) = \begin{cases} e^{-1/t} & \text{if } t > 0 \\ 0 & \text{if } t \le 0 \end{cases} ψ(t)={e−1/t0​if t>0if t≤0​

This function is famously C∞C^\inftyC∞ on all of R\mathbb{R}R, including at t=0t=0t=0 where all its derivatives are zero. Using this, we can construct a smooth function on Rn\mathbb{R}^nRn that is positive on an open ball and zero elsewhere. For example, the function Ψ:Rn→R\Psi: \mathbb{R}^n \to \mathbb{R}Ψ:Rn→R defined by

Ψ(x)={e−1/(1−∥x∥2)if ∥x∥<10if ∥x∥≥1\Psi(x) = \begin{cases} e^{-1/(1-\|x\|^2)} & \text{if } \|x\| < 1 \\ 0 & \text{if } \|x\| \ge 1 \end{cases} Ψ(x)={e−1/(1−∥x∥2)0​if ∥x∥<1if ∥x∥≥1​

is a smooth function that is positive on the open unit ball B(0,1)B(0,1)B(0,1) and has its support contained in the closed unit ball B(0,1)‾\overline{B(0,1)}B(0,1)​.

By scaling and translating this function, we can create a bump function for any ball. More generally, we have the following crucial existence theorem for bump functions on manifolds:

Theorem (Existence of Bump Functions): Let MMM be a smooth manifold, K⊂MK \subset MK⊂M be a compact set, and U⊂MU \subset MU⊂M be an open set containing KKK. Then there exists a smooth function λ:M→[0,1]\lambda: M \to [0, 1]λ:M→[0,1] such that:

  1. λ(p)=1\lambda(p) = 1λ(p)=1 for all p∈Kp \in Kp∈K.
  2. supp(λ)⊂U\text{supp}(\lambda) \subset Usupp(λ)⊂U.

Here, the support of a function f:M→Rf: M \to \mathbb{R}f:M→R, denoted supp(f)\text{supp}(f)supp(f), is the closure of the set of points where fff is non-zero:

supp(f)={p∈M∣f(p)≠0}‾\text{supp}(f) = \overline{\{p \in M \mid f(p) \neq 0\}} supp(f)={p∈M∣f(p)=0}​

A function with compact support is a function whose support is a compact set.

Definition of a Partition of Unity

Let MMM be a smooth manifold and let U={Uα}α∈A\mathcal{U} = \{U_\alpha\}_{\alpha \in A}U={Uα​}α∈A​ be an open cover of MMM. A family of smooth functions {ρα:M→[0,1]}α∈A\{\rho_\alpha: M \to [0,1]\}_{\alpha \in A}{ρα​:M→[0,1]}α∈A​ indexed by the same set AAA is called a partition of unity subordinate to the cover U\mathcal{U}U if it satisfies the following conditions:

  1. Subordination: For each α∈A\alpha \in Aα∈A, the support of ρα\rho_\alphaρα​ is contained in UαU_\alphaUα​. That is, supp(ρα)⊂Uα\text{supp}(\rho_\alpha) \subset U_\alphasupp(ρα​)⊂Uα​.
  2. Local Finiteness: The cover of supports {supp(ρα)}α∈A\{\text{supp}(\rho_\alpha)\}_{\alpha \in A}{supp(ρα​)}α∈A​ is locally finite. This means that for every point p∈Mp \in Mp∈M, there exists a neighborhood VVV of ppp that intersects only a finite number of the sets supp(ρα)\text{supp}(\rho_\alpha)supp(ρα​).
  3. Sum to Unity: For every point p∈Mp \in Mp∈M, the sum of the function values at that point is 1.

    ∑α∈Aρα(p)=1\sum_{\alpha \in A} \rho_\alpha(p) = 1 α∈A∑​ρα​(p)=1

    (The local finiteness condition ensures that for any ppp, this is a finite sum in a neighborhood of ppp, and thus the total sum is a well-defined smooth function.)

Existence of Partitions of Unity

The main theorem guarantees that such partitions of unity always exist on manifolds that satisfy the standard topological assumptions.

Theorem (Existence of Partitions of Unity): Let MMM be a smooth manifold (which is Hausdorff and second-countable by our definition). For any open cover U\mathcal{U}U of MMM, there exists a smooth partition of unity subordinate to U\mathcal{U}U.

Sketch of Proof:
The proof relies on the topological property of paracompactness, which is guaranteed for Hausdorff, second-countable manifolds.

  1. Find a good refinement: Since MMM is second-countable and locally compact, we can find a countable, locally finite open refinement V={Vj}j=1∞\mathcal{V} = \{V_j\}_{j=1}^\inftyV={Vj​}j=1∞​ of the original cover U\mathcal{U}U such that each Vj‾\overline{V_j}Vj​​ is compact and for each jjj, there is some αj\alpha_jαj​ with Vj‾⊂Uαj\overline{V_j} \subset U_{\alpha_j}Vj​​⊂Uαj​​.
  2. Find another refinement: We can construct another open cover W={Wj}j=1∞\mathcal{W} = \{W_j\}_{j=1}^\inftyW={Wj​}j=1∞​ such that for each jjj, Wj‾\overline{W_j}Wj​​ is compact and Wj‾⊂Vj\overline{W_j} \subset V_jWj​​⊂Vj​.
  3. Construct bump functions: For each jjj, since Wj‾\overline{W_j}Wj​​ is a compact set contained in the open set VjV_jVj​, we can use the bump function existence theorem to find a smooth function ψj:M→[0,1]\psi_j: M \to [0,1]ψj​:M→[0,1] such that ψj≡1\psi_j \equiv 1ψj​≡1 on Wj‾\overline{W_j}Wj​​ and supp(ψj)⊂Vj\text{supp}(\psi_j) \subset V_jsupp(ψj​)⊂Vj​.
  4. Sum the bump functions: Define a function Ψ:M→R\Psi: M \to \mathbb{R}Ψ:M→R by Ψ(p)=∑j=1∞ψj(p)\Psi(p) = \sum_{j=1}^\infty \psi_j(p)Ψ(p)=∑j=1∞​ψj​(p). Since the cover {Vj}\{V_j\}{Vj​} (and thus the supports of the ψj\psi_jψj​) is locally finite, this sum is finite in a neighborhood of any point, so Ψ\PsiΨ is a smooth function. Since {Wj}\{W_j\}{Wj​} is a cover, for any p∈Mp \in Mp∈M, p∈Wjp \in W_jp∈Wj​ for some jjj, which means ψj(p)=1\psi_j(p) = 1ψj​(p)=1. Therefore, Ψ(p)>0\Psi(p) > 0Ψ(p)>0 for all p∈Mp \in Mp∈M.
  5. Normalize: For each jjj, define ρj(p)=ψj(p)Ψ(p)\rho_j(p) = \frac{\psi_j(p)}{\Psi(p)}ρj​(p)=Ψ(p)ψj​(p)​. This family {ρj}j=1∞\{\rho_j\}_{j=1}^\infty{ρj​}j=1∞​ is a partition of unity subordinate to the cover V\mathcal{V}V, and therefore also subordinate to the original cover U\mathcal{U}U.

Applications of Partitions of Unity

Partitions of unity are essential for extending local properties to global ones. Here are two classic applications:

  • Integration on Manifolds: To define the integral of a function fff on a manifold MMM, one can use a partition of unity {ρα}\{\rho_\alpha\}{ρα​} subordinate to a cover of coordinate charts {Uα}\{U_\alpha\}{Uα​}. The integral is then defined as a sum of integrals over Euclidean space:

    ∫Mf dV=∑α∫Uα(ραf) dV=∑α∫ϕα(Uα)(ραf)∘ϕα−1 dx1⋯dxn\int_M f \, dV = \sum_\alpha \int_{U_\alpha} (\rho_\alpha f) \, dV = \sum_\alpha \int_{\phi_\alpha(U_\alpha)} (\rho_\alpha f) \circ \phi_\alpha^{-1} \, dx^1 \cdots dx^n ∫M​fdV=α∑​∫Uα​​(ρα​f)dV=α∑​∫ϕα​(Uα​)​(ρα​f)∘ϕα−1​dx1⋯dxn

    The partition of unity ensures that each piece (ραf)(\rho_\alpha f)(ρα​f) is compactly supported within a single chart, making the integral well-defined, and that the sum captures the “whole” of fff.

  • Existence of Riemannian Metrics: Any smooth manifold admits a Riemannian metric. To prove this, one can define a Euclidean metric on each coordinate chart (Uα,ϕα)(U_\alpha, \phi_\alpha)(Uα​,ϕα​). Using a partition of unity {ρα}\{\rho_\alpha\}{ρα​} subordinate to the chart cover, one can patch these local metrics together into a global metric ggg via a weighted sum:

    g=∑αραgαg = \sum_\alpha \rho_\alpha g_\alpha g=α∑​ρα​gα​

    where gαg_\alphagα​ is the pullback of the Euclidean metric from Rn\mathbb{R}^nRn to UαU_\alphaUα​. The result is a smoothly varying inner product on each tangent space, i.e., a Riemannian metric.

First, we denote R+n={(x1,…,xn)∈Rn:xn≥0}\mathbb R^n_+ = \{(x^1, \ldots, x^n) \in \mathbb R^n : x^n \geq 0\}R+n​={(x1,…,xn)∈Rn:xn≥0} as the half-space in Rn\mathbb R^nRn.
