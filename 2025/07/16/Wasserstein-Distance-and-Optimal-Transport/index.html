Wasserstein Distance and Optimal Transport

Optimal Transport Problem

Definition

Given two probability measures μ\muμ and ν\nuν on measurable spaces (X,A)(X, \mathcal{A})(X,A) and (Y,B)(Y, \mathcal{B})(Y,B), respectively, the Optimal Transport Problem seeks to find a transport plan π\piπ that minimizes the cost of transporting mass from μ\muμ to ν\nuν. The cost function is typically defined as a function c:X×Y→Rc: X \times Y \to \mathbb{R}c:X×Y→R, which quantifies the cost of moving a unit mass from point x∈Xx \in Xx∈X to point y∈Yy \in Yy∈Y.

The problem can be formulated as follows (Monge’s formulation):

inf⁡T∫Xc(x,T(x)) dμ(x)\begin{aligned} \inf_{T} \int_{X} c(x, T(x)) \, d\mu(x) \end{aligned} Tinf​∫X​c(x,T(x))dμ(x)​

where T:X→YT: X \to YT:X→Y is a transport map that pushes μ\muμ forward to ν\nuν, i.e., T∗μ=νT_* \mu = \nuT∗​μ=ν.

Such a transport map TTT is called an optimal transport map if it minimizes the cost function, although it is not always guaranteed to exist since the pushforward is not necessarily possible.

More generally, we can express the problem as (Kantorovich’s formulation):

inf⁡π∈Π(μ,ν)∫X×Yc(x,y) dπ(x,y)\begin{aligned} \inf_{\pi \in \Pi(\mu, \nu)} \int_{X \times Y} c(x, y) \, d\pi(x, y) \end{aligned} π∈Π(μ,ν)inf​∫X×Y​c(x,y)dπ(x,y)​

where Π(μ,ν)\Pi(\mu, \nu)Π(μ,ν) is the set of all joint distributions of μ\muμ and ν\nuν.

Dual Formulation

The primal problem is a convex optimization problem since it is linear w.r.t. π\piπ and the feasible set is convex.

The constraint is that π\piπ must be a joint distribution of μ\muμ and ν\nuν, which can be expressed as:

∫X×Yf(x) dγ(x,y)=∫Xf(x) dμ(x),∀f∈C(X)∫X×Yg(y) dγ(x,y)=∫Yg(y) dν(y),∀g∈C(Y)\begin{aligned} \int_{X \times Y} f(x) \, d\gamma(x, y) &= \int_X f(x) \, d\mu(x), \quad \forall f \in C(X)\\ \int_{X \times Y} g(y) \, d\gamma(x, y) &= \int_Y g(y) \, d\nu(y), \quad \forall g \in C(Y) \end{aligned} ∫X×Y​f(x)dγ(x,y)∫X×Y​g(y)dγ(x,y)​=∫X​f(x)dμ(x),∀f∈C(X)=∫Y​g(y)dν(y),∀g∈C(Y)​

where C(X)C(X)C(X) and C(Y)C(Y)C(Y) are the spaces of continuous functions on XXX and YYY, respectively.

This form allows us to derive the dual formulation of the optimal transport problem, whose Lagrangian is given by:

L(π,λ,μ)=∫X×Yc(x,y)dπ+∫Xλ(x)(dμ−Proj⁡X(dπ))+∫Yμ(y)(dν−Proj⁡Y(dπ))=∫X×Y(c(x,y)−λ(x)−μ(y))dπ+∫Xλ(x)dμ+∫Yμ(y)dν\begin{aligned} L(\pi, \lambda, \mu) &= \int_{X \times Y} c(x,y) d\pi + \int_{X} \lambda(x) ( d\mu -\operatorname{Proj}_X(d\pi)) + \int_{Y} \mu(y) (d\nu - \operatorname{Proj}_Y(d\pi)) \\ &= \int_{X \times Y} \left ( c(x,y) - \lambda(x) - \mu(y) \right ) d\pi + \int_{X} \lambda(x) d\mu + \int_{Y} \mu(y) d\nu \end{aligned} L(π,λ,μ)​=∫X×Y​c(x,y)dπ+∫X​λ(x)(dμ−ProjX​(dπ))+∫Y​μ(y)(dν−ProjY​(dπ))=∫X×Y​(c(x,y)−λ(x)−μ(y))dπ+∫X​λ(x)dμ+∫Y​μ(y)dν​

Thus the dual problem can be expressed as:

sup⁡λ,μinf⁡π∈Π(μ,ν)L(π,λ,μ)\begin{aligned} \sup_{\lambda, \mu} \inf_{\pi \in \Pi(\mu, \nu)} L(\pi, \lambda, \mu) \end{aligned} λ,μsup​π∈Π(μ,ν)inf​L(π,λ,μ)​

For any point (x,y)(x,y)(x,y), if c(x,y)−λ(x)−μ(y)<0c(x,y) - \lambda(x) - \mu(y) < 0c(x,y)−λ(x)−μ(y)<0, then the value of L(π,λ,μ)L(\pi, \lambda, \mu)L(π,λ,μ) can be made arbitrarily to −∞-\infty−∞ by increasing π\piπ.

Therefore, the optimal solution must satisfy:

c(x,y)≥λ(x)+μ(y),∀(x,y)∈X×Yc(x,y) \geq \lambda(x) + \mu(y), \quad \forall (x,y) \in X \times Y c(x,y)≥λ(x)+μ(y),∀(x,y)∈X×Y

And the dual problem now becomes:

sup⁡λ,μ{∫Xλ(x)dμ+∫Yμ(y)dν:c(x,y)≥λ(x)+μ(y),∀(x,y)∈X×Y}\begin{aligned} \sup_{\lambda, \mu} \left \{ \int_{X} \lambda(x) d\mu + \int_{Y} \mu(y) d\nu : c(x,y) \geq \lambda(x) + \mu(y), \forall (x,y) \in X \times Y \right \} \end{aligned} λ,μsup​{∫X​λ(x)dμ+∫Y​μ(y)dν:c(x,y)≥λ(x)+μ(y),∀(x,y)∈X×Y}​

Or more compactly:

sup⁡λ,μ{∫Xλ(x)dμ+∫Yμ(y)dν:λ⊕μ≤c}\begin{aligned} \sup_{\lambda, \mu} \left \{ \int_{X} \lambda(x) d\mu + \int_{Y} \mu(y) d\nu : \lambda \oplus \mu \leq c \right \} \end{aligned} λ,μsup​{∫X​λ(x)dμ+∫Y​μ(y)dν:λ⊕μ≤c}​

The strong duality holds under mild conditions, such XXX and YYY are compact and ccc is continuous. More generally, ccc can be a lower semicontinuous function and exist a feasible π\piπ with finite cost.

Wasserstein Distance

When X=Y,c(x,y)=d(x,y)X = Y, c(x,y) = d(x,y)X=Y,c(x,y)=d(x,y) is a metric, the optimal transport problem induces the Wasserstein distance:

Wp(μ,ν)=inf⁡π∈Π(μ,ν)(∫X×Yd(x,y)p dπ(x,y))1/p\begin{aligned} W_p(\mu, \nu) = \inf_{\pi \in \Pi(\mu, \nu)} \left( \int_{X \times Y} d(x,y)^p \, d\pi(x,y) \right)^{1/p} \end{aligned} Wp​(μ,ν)=π∈Π(μ,ν)inf​(∫X×Y​d(x,y)pdπ(x,y))1/p​

This distance is a metric on the space of probability measures with finite ppp-th moment, and it satisfies the properties of a metric:

  1. Non-negativity: Wp(μ,ν)≥0W_p(\mu, \nu) \geq 0Wp​(μ,ν)≥0 for all μ,ν\mu, \nuμ,ν.
  2. Identity of indiscernibles: Wp(μ,ν)=0W_p(\mu, \nu) = 0Wp​(μ,ν)=0 if and only if μ=ν\mu = \nuμ=ν.
  3. Symmetry: Wp(μ,ν)=Wp(ν,μ)W_p(\mu, \nu) = W_p(\nu, \mu)Wp​(μ,ν)=Wp​(ν,μ).
  4. Triangle inequality: Wp(μ,ν)+Wp(ν,σ)≥Wp(μ,σ)W_p(\mu, \nu) + W_p(\nu, \sigma) \geq W_p(\mu, \sigma)Wp​(μ,ν)+Wp​(ν,σ)≥Wp​(μ,σ) for all μ,ν,σ\mu, \nu, \sigmaμ,ν,σ.
