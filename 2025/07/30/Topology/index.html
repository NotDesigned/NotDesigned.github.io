Basic Concepts

Topology Space

A topology space is a set XXX with a collection of open sets T\mathcal{T}T that satisfy the following properties:

  1. ∅∈T\emptyset \in \mathcal{T}∅∈T and X∈TX \in \mathcal{T}X∈T.

  3. if U1,U2∈TU_1, U_2 \in \mathcal{T}U1​,U2​∈T, then U1∩U2∈TU_1 \cap U_2 \in \mathcal{T}U1​∩U2​∈T.

A topological space is denoted as (X,T)(X, \mathcal{T})(X,T).

  1. For each x∈Xx \in Xx∈X, there exists a basis element B∈BB \in \mathcal{B}B∈B such that x∈Bx \in Bx∈B.
  2. If B1,B2∈BB_1, B_2 \in \mathcal{B}B1​,B2​∈B and x∈B1∩B2x \in B_1 \cap B_2x∈B1​∩B2​, then there exists a basis element B3∈BB_3 \in \mathcal{B}B3​∈B such that x∈B3⊆B1∩B2x \in B_3 \subseteq B_1 \cap B_2x∈B3​⊆B1​∩B2​.

Countability, Compactness and Separation Axioms

A topological space is first countable if every point has a countable local base, meaning for each point x∈Xx \in Xx∈X, there exists a countable collection of open sets {Un}n=1∞\{U_n\}_{n=1}^{\infty}{Un​}n=1∞​ such that for any open set UUU containing xxx, there exists some nnn such that Un⊆UU_n \subseteq UUn​⊆U.

A topological space is second countable if it has a countable base, meaning there exists a countable collection of open sets B\mathcal{B}B such that every open set in the topology can be expressed as a union of sets from B\mathcal{B}B.

A topological space is compact if every open cover has a finite subcover, meaning for any collection of open sets {Ui}i∈I\{U_i\}_{i \in I}{Ui​}i∈I​ such that X=⋃i∈IUiX = \bigcup_{i \in I} U_iX=⋃i∈I​Ui​, there exists a finite subset J⊆IJ \subseteq IJ⊆I such that X=⋃j∈JUjX = \bigcup_{j \in J} U_jX=⋃j∈J​Uj​.

A topological space is T0T_0T0​ (Kolmogorov) if for any two distinct points x,y∈Xx, y \in Xx,y∈X, there exists an open set containing one of the points but not the other. Intuitively, this means that a pair of point can be one-sidely distinguished by open sets for one of them.

A topological space is T1T_1T1​ (Frechet) if for any two distinct points x,y∈Xx, y \in Xx,y∈X, there exist open sets UxU_xUx​ and UyU_yUy​ such that x∈Uxx \in U_xx∈Ux​ and y∉Uxy \notin U_xy∈/Ux​, and y∈Uyy \in U_yy∈Uy​ and x∉Uyx \notin U_yx∈/Uy​. Intuitively, this means that points can be one-sidely separated by open sets for both of them.

A topological space is T2T_2T2​ (Hausdorff) if for any two distinct points x,y∈Xx, y \in Xx,y∈X, there exist disjoint open sets UxU_xUx​ and UyU_yUy​ such that x∈Uxx \in U_xx∈Ux​ and y∈Uyy \in U_yy∈Uy​. This means that points can be separated by disjoint open sets.

A topological space is regular if it is T1T_1T1​ and for every point xxx and closed set FFF not containing xxx, there exist disjoint open sets UUU and VVV such that x∈Ux \in Ux∈U and F⊆VF \subseteq VF⊆V. This means that points can be separated from closed sets by disjoint open sets.

A topological space is normal if for any two disjoint closed sets AAA and BBB, there exist disjoint open sets UUU and VVV such that A⊆UA \subseteq UA⊆U and B⊆VB \subseteq VB⊆V. This means that closed sets can be separated by disjoint open sets.

A topological space is T3T_3T3​ (regular Hausdorff) if it is both regular and Hausdorff. This means that points can be separated from closed sets by disjoint open sets, and distinct points can be separated by disjoint open sets. T3T_3T3​ spaces are also normal.

Important properties of these separation axioms include:

  • T0⊆T1⊆T2⊆T3T_0 \subseteq T_1 \subseteq T_2 \subseteq T_3T0​⊆T1​⊆T2​⊆T3​.
  • A single point set is closed in a T1T_1T1​ space.
  • A T2T_2T2​ space is normal if it is also regular since a point is closed in a T2T_2T2​ space.

Connectedness and Path Connectedness

A topological space is connected if it cannot be expressed as the union of two disjoint non-empty open sets. In other words, there are no two open sets UUU and VVV such that X=U∪VX = U \cup VX=U∪V, U∩V=∅U \cap V = \emptysetU∩V=∅, and both UUU and VVV are non-empty.

A topological space is path connected if for any two points x,y∈Xx, y \in Xx,y∈X, there exists a continuous function (path) f:[0,1]→Xf: [0, 1] \to Xf:[0,1]→X such that f(0)=xf(0) = xf(0)=x and f(1)=yf(1) = yf(1)=y. This means that there is a continuous path connecting any two points in the space.

for each nnn, UnU_nUn​ is path connected and contains xxx, and for any open set UUU containing xxx, there exists some nnn such that Un⊆UU_n \subseteq UUn​⊆U.

A topological space is locally connected if every point has a neighborhood base consisting of connected sets. This means that for each point x∈Xx \in Xx∈X, there exists a collection of open sets {Un}n=1∞\{U_n\}_{n=1}^{\infty}{Un​}n=1∞​ such that for each nnn, UnU_nUn​ is connected and contains xxx, and for any open set UUU containing xxx, there exists some nnn such that Un⊆UU_n \subseteq UUn​⊆U.

Common Topologies

Fineness and coarseness of topologies are important concepts in topology. A topology T1\mathcal{T}_1T1​ is finer than another topology T2\mathcal{T}_2T2​ on the same set XXX if every open set in T2\mathcal{T}_2T2​ is also an open set in T1\mathcal{T}_1T1​. In other words, T1⊇T2\mathcal{T}_1 \supseteq \mathcal{T}_2T1​⊇T2​. Conversely, T2\mathcal{T}_2T2​ is coarser than T1\mathcal{T}_1T1​ if T1\mathcal{T}_1T1​ is finer than T2\mathcal{T}_2T2​.

The discrete topology on a set XXX is the finest topology, where every subset of XXX is open. Formally, Tdiscrete=P(X)\mathcal{T}_{\text{discrete}} = \mathcal{P}(X)Tdiscrete​=P(X), the power set of XXX.

The indiscrete topology (or trivial topology) on a set XXX is the coarsest topology, where only the empty set and the entire set XXX are open. Formally, Tindiscrete={∅,X}\mathcal{T}_{\text{indiscrete}} = \{\emptyset, X\}Tindiscrete​={∅,X}.

The quotient topology on a set XXX with an equivalence relation ∼\sim∼ is defined as follows:

  1. The quotient set is X/∼={[x]:x∈X}X / \sim = \{ [x] : x \in X \}X/∼={[x]:x∈X}, where [x][x][x] is the equivalence class of xxx.
  2. A subset U⊆X/∼U \subseteq X / \simU⊆X/∼ is open in the quotient topology if and only if its preimage under the natural projection map π:X→X/∼\pi: X \to X / \simπ:X→X/∼ is open in XXX. That is, UUU is open if π−1(U)\pi^{-1}(U)π−1(U) is open in XXX.

In other words, the quotient topology is the coarsest topology on X/∼X / \simX/∼ such that the natural projection map π\piπ is continuous.

  2. A subset U⊆XU \subseteq XU⊆X is open in the product topology if it can be expressed as a union of sets of the form ∏i∈IUi\prod_{i \in I} U_i∏i∈I​Ui​, where UiU_iUi​ is open in XiX_iXi​ for each i∈Ii \in Ii∈I, and Ui=XiU_i = X_iUi​=Xi​ for all but finitely many iii.

  1. The box space is X=∏i∈IXiX = \prod_{i \in I} X_iX=∏i∈I​Xi​.
  2. A subset U⊆XU \subseteq XU⊆X is open in the box topology if it can be expressed as a union of sets of the form ∏i∈IUi\prod_{i \in I} U_i∏i∈I​Ui​, where UiU_iUi​ is open in XiX_iXi​ for each i∈Ii \in Ii∈I.
    In other words, the box topology is generated by the basis consisting of all products of open sets, where each factor can be any open set in the corresponding space.
